Crash Numbers
Dodge Zeros

In "Countdown" you control 'P' among 70 numbers. Your mission is to push and strategically maneuver these numbers, earning points by smashing them against walls. But beware, within this dynamic labyrinth, zeros roam unpredictably, threatening 'P'. An encounter with these zeroes will cost 'P' a life.

More Info

About The Game

Aim of the game "Countdown" is collecting treasures while walking in the maze. In this game treasures are numbers. The game is played in a 23x53 game field including outer walls. Game characters are 'P' and numbers. Main mechanic of the game is countdown mechanic. Countdown action is happening in every 15 seconds. And countdown basically decreases the values of every number except 0 and 1. Number 1 has a %3 chance of turning into number 0 and become alive.


Game Characters

P represents the human player who plays the game.

'P' is moved via cursor keys.

At the beginning human player has 5 lives

Numbers which are generated randomly is from 0 to 9.

Number 0 is alive can move randomly in 4 directions. Other numbers are static


Game Initialization

The inner walls in the game area are generated in random places

There are 3 types of inner walls:

-- 3x Long Wall which has length of 11

-- 5x Medium Wall which has length of 7

-- 20x Short Wall which has length of 3


Human player 'P' is located randomly on an empty space.

Numbers are also randomly generated on empty spaces.

Game Rules

-- Displayed time unit in the game is approximately 1 second

-- If number 0 and 'P' encounters on the same square 'P' loses a life

-- If 'P' loses all 5 of it's lives the game is over

-- 'P' can push non-increasing number sequences, if there is an empty square behind the number

-- if there is a wall behind the last number 'P' can smash the last number when pushing non-increasing number sequences. P cannot smash the number in the neighboring square. Non-pushable sequences are also non-smashable.

-- For each smashed number, a new random number (5, 6, 7, 8 or 9 with equal probability) is generated in a random place.

-- Smashing 0 will gain player 20 points.

-- Smashing 1,2,3 and 4 will gain player 2 points.

-- Smashing 5,6,7,8 and 9 will gain player 1 points.
